## Features

- [x] Basic routing
- [x] Authentication using devise
- [x] Test publishing on Heroku

## Stack used

- Devise `4.3.0`
- Rails `5.1.4`
- Ruby `2.4.2`

## Getting started

#### Requirements

- `bundler`
- `rails`
- Ruby 2.4+

#### Installation

```sh
git clone https://github.com/jclerc/learning-rails.git
cd learning-rails
# install dependencies
bundle install
# start server
rails server
```
